## Setup and Running

> - Clone this repo: `$ git clone https://github.com/vovoka-path/rs-clone.git`.
> - Go to branch: `git checkout crm`
> - Go to folder: `$ cd crm`.
> - Install dependencies: `$ npm install`.
> - Start server: `$ npm start`.
